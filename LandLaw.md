# 土地法学复习讲义

## 一、核心知识点梳理

### 1. 土地权利主体与义务主体
- **土地使用权法律关系的权利主体**：企事业单位、国家机关和其他社会组织（同时也是土地管理和土地所有权关系的义务主体）

### 2. 农村集体建设用地使用原则
- 必须用于乡镇企业、乡(镇)村公共设施、公益事业、农村村民住宅等乡(镇)村建设项目
- 必须符合乡(镇)土地利用总体规划和土地利用年度计划
- 必须依法取得**县级以上**人民政府批准（非市级以上）
- 建设占地与农村土地整理挂钩，严格控制占用耕地

### 3. 土地使用权期限规定
- **工业用地使用权出让最高年限**：50年
- **临时用地期限**：不得超过2年，超过需征得原使用单位同意
- **耕地承包期**：30年（《农地承包法》第20条）
- **林地承包期**：30-70年

### 4. 物权类型与特征
- **地役权**：为了自己土地的通行便利，在他人土地上修造道路的物权
- **抵押权人最基本权利**：优先受偿权
- **用益物权与担保物权区别**：
  - 对标的物支配方面不同
  - 权利实现时间不同
  - 用益物权为独立物权，担保物权为从属物权（题目中C选项表述错误）
  - 物上代位性不同

### 5. 土地所有权制度
- **农民集体所有土地管理**：由村集体经济组织或村民委员会经营、管理（《土地管理法》规定）
- **土地国家所有权建立**：1956年完成生产资料私有制社会主义改造后逐步建立
- **划拨土地使用权特点**：无期限、无地租
- **土地使用权划拨性质**：行政行为

## 二、重要法律制度

### 1. 耕地保护制度
- **省级人民政府职责**：
  - 制定开垦耕地计划
  - 对建设单位和县、市人民政府开垦耕地进行监督
  - 组织开垦耕地
- **不包括**：监督和验收城市建设用地区外建设项目用地

### 2. 征地补偿制度
- **征地补偿范围**：
  - 土地补偿费
  - 青苗补偿费
  - 地上附着物补偿费
  - 安置补助费
- **补偿原则**：
  - 补偿性质而非地价
  - 保证被征地农民生活水平不降低
  - 按原有用途补偿
  - 依照法定标准补偿

### 3. 土地有偿使用制度形式
- 土地使用权出让
- 土地使用权租赁
- 土地使用权作价入股
- 土地承包经营

### 4. 土地用途分类
- 农业用地
- 建设用地
- 未利用土地

## 三、程序性规定

### 1. 行政复议
- **性质**：依申请产生的具体行政行为
- **受理机关**：国家行政机关（上一级行政机关）
- **申请期限**：自知道具体行政行为之日起60日内

### 2. 土地行政执法生效要件
- **主体要件**：县级以上人民政府地政部门及其授权或委托部门
- **内容要件**：在法律赋予的权限范围内行使执法权
- **程序要件**：符合法定程序（如出示土地监察证件、办理农用地转用审批手续等）

### 3. 具体时限规定
- **划拨土地补办出让手续审批**：15日内书面函告
- **土地使用权出让金支付**：签订合同后60日内
- **征地批准权限**：省级政府可批准征用基本农田以外耕地低于35公顷

## 四、概念解析

### 1. 农用地转用
农用地转为建设用地的简称，指将耕地、林地、草地等农业生产用地，按土地利用总体规划和国家规定批准权限报批后，转变为建造建筑物、构筑物的土地的行为。

### 2. 农村集体建设用地
农民集体所有的用于建造建筑物、构筑物的土地，包括原有的建设用地和经批准办理农用地转用手续的农用地。

### 3. 国有土地使用权出让
国家以土地所有者身份将土地使用权在一定年限内让与土地使用者，并由土地使用者向国家交付土地使用权出让金的行为。

### 4. 土地行政复议
土地管理特定部门内，针对土地具体行政行为（或土地执法行为）而产生的行政复议。

## 五、拓展知识

### 1. 用益物权体系
我国用益物权体系主要包括：
- 农地使用权
- 基地使用权  
- 地役权

### 2. 物权排他性效力表现
- 同一标的物上不得同时并立两个所有权
- 同一标的物上不得同时成立两个或以上基地使用权或农地使用权
- 用益物权与担保物权能同时并存
- 抵押权等担保物权可复数同时存在，效力按先后顺序确定
- 基地使用权与抵押权能同时并存

### 3. 国有土地使用权转让与出让区别
| 对比维度 | 出让 | 转让 |
|---------|------|------|
| 行为性质 | 创设财产权 | 转移财产权 |
| 主体关系 | 土地所有者vs使用者 | 使用者vs使用者 |
| 费用构成 | 主要是地租 | 包含出让金、投资、利润等 |
| 对所有权影响 | 创设用益物权 | 不影响国家所有权 |

### 4. 土地法律关系保护机构
- 中央和地方各级政府地政部门
- 乡级人民政府  
- 司法机关
